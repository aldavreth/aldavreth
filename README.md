### 👋 About Me

I'm **Ryan**, a Backend & Cloud Developer based in Pasuruan, Indonesia 🇮🇩

I specialize in designing and building scalable, cloud-native backend systems, from API architecture to deploying and orchestrating containerized services on AWS and Google Cloud. Most of my work centers on backend architecture that is efficient, reliable, and built to last.

Right now I'm deepening my expertise in AWS and Google Cloud infrastructure, and building AI-integrated applications and automation systems. Outside of work, I'm usually listening to Moonlight Sonata. I believe great software should be fast, minimal, and purposeful.
